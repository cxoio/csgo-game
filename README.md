# CS:GO Game Documentation

## Game Features
- Realistic graphics and physics.
- Multiple game modes including Deathmatch, Casual, and Competitive.
- A variety of weapons and equipment to choose from.
- Social features including friend lists and matchmaking.

## Controls
- **WASD** for movement.
- **Mouse Look** for camera control.
- **LMB (Left Mouse Button)** to shoot.
- **RMB (Right Mouse Button)** to aim down sights.

## Installation Instructions
1. Download the game from the official website or store.
2. Install the game by following the on-screen instructions.
3. Launch the game from your application menu or desktop shortcut.

## Gameplay Overview
In CS:GO, players compete in teams to complete objectives and defeat their opponents. Experience intense battles across various map environments.

## Technical Details
- Developed in C++ and utilizing the Source engine.
- Supports multiplayer online gameplay.
- Frequent updates for game balance and new content.
